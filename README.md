An AI-powered web application that summarizes YouTube videos instantly using transcript extraction and Large Language Models (LLMs).

This tool allows users to paste a YouTube video URL and quickly get a clear and easy-to-understand summary of the video content.

### 🧠 How It Works

1. User enters a YouTube video URL
2. The system extracts the video transcript
3. The transcript is processed using an AI model
4. The AI generates a concise summary
5. The summary is displayed in the Streamlit interface

- **Python**
- **Streamlit** – Web app interface
- **LangChain** – LLM workflow framework
- **Google Gemini API** – AI model for summarization
- **YouTube Transcript API** – Extracts video transcripts

## ⚙️ Installation

### 1️⃣ Clone the repository

git clone:  https://github.com/Mohamed-Imraan/youtube-video-summarizer.git

### 2️⃣ Navigate to the project folder

cd youtube-video-summarizer

### 3️⃣ Install dependencies

pip install -r requirements.txt

### 4️⃣ Run the application

streamlit run summarizer.py

---

### 🔑 Environment Variables

Create a ".env" file in the project folder and add your API key:

GOOGLE_API_KEY=your_api_key_here

---

### ⚠️ Disclaimer

This project uses the Google Gemini API to generate summaries.

The free API tier has limited usage quotas.
If the quota limit is reached, the summarization output may not appear.

To run the project successfully, please use a valid API key with sufficient quota.
